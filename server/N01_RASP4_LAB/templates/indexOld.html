Sobre o GreenSe

O GreenSe é um sistema de monitoramento e controle para estufas agrícolas, utilizando sensores e IoT para garantir um ambiente otimizado para o cultivo.

Como Funciona

Os dados coletados por sensores são enviados para um sistema central, analisados e exibidos nos dashboards do Grafana para fácil visualização e tomada de decisão.
